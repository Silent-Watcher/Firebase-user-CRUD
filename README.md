## Run Locally

Clone the project

```bash
  git clone https://github.com/Silent-Watcher/Firebase-user-CRUD
```

Go to the project directory

```bash
  cd Firebase-user-CRUD
```

Install dependencies

```bash
  npm install
  yarn install
```

Start the server

```bash
  npm run dev
  yarn dev
```
